Class和普通构造函数有何区别

我们经常会用ES6中的Class来代替JS中的构造函数做开发。

  • Class 在语法上更加贴合面向对象的写法

  • Class 实现继承更加易读、易理解

  • 更易于写 java 等后端语言的使用
  • 本质还是语法糖，使用 prototype
