AMD的基本语法

AMD的概念

AMD（Asynchronous Module Definition）：异步模块定义。AMD专门用于浏览器端，模块的加载是异步的。

AMD规范：是 RequireJS 在推广过程中对模块化定义的规范化产出。

RequireJS：一个基于AMD规范实现的模块化开发解决方案。
