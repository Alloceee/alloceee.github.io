前言

ECMAScript 是 JS 的语言标准。而 ES6 是新的 JS 语法标准。

发展历史

20180303_1633.png

  • 2015年6月，ES6正式发布。

ES6 的其他优势

  • 使用 babel 语法转换器，支持低端浏览器

  • 流行的库基本都是基于 ES6 构建。 React 默认使用 ES6 新源发开发。

ES6 的常用语法

ES6语法概览

  • 块级作用域、字符串

  • 对象扩展、解构

  • 类、模块化等。

作用域：let 和 const

  • 用 let定义变量 ，替代 var

  • 用const 定义常量（定义后，不可修改）

  • 作用域和 {}
