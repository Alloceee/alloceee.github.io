常见的状态码

  • 200 OK 客户端请求成功
  • 301 Moved Permanently （永久移除），请求的URL已移走。Response中应该包含一个Location URL,说明资源现在所处的位置
  • 302 found 重定向
  • 400 Bad Request 客户端请求有语法错误，不能被服务器解析
  • 404 表示没有找到请求的资源
  • 500 表示服务端内部错误
